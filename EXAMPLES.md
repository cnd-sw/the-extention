# Grammar Correction Examples

The extension now corrects REAL grammar mistakes! Here are examples:

##  Spelling Corrections

| Before | After |
|--------|-------|
| `teh cat is here` | `The cat is here.` |
| `recieve the package` | `Receive the package.` |
| `seperate items` | `Separate items.` |
| `occured yesterday` | `Occurred yesterday.` |

##  Contractions & Informal Language

| Before | After |
|--------|-------|
| `im gonna do it` | `I am going to do it.` |
| `u cant do that` | `You cannot do that.` |
| `idk what to do` | `I do not know what to do.` |
| `wanna go there` | `Want to go there.` |
| `its gonna be great` | `It is going to be great.` |

##  Subject-Verb Agreement

| Before | After |
|--------|-------|
| `he are going` | `He is going.` |
| `they is here` | `They are here.` |
| `she are happy` | `She is happy.` |
| `I is ready` | `I am ready.` |

##  Articles (a vs an)

| Before | After |
|--------|-------|
| `a apple` | `An apple.` |
| `an cat` | `A cat.` |
| `a elephant` | `An elephant.` |
| `an book` | `A book.` |

##  Capitalization

| Before | After |
|--------|-------|
| `i am here` | `I am here.` |
| `hello. how are you` | `Hello. How are you.` |
| `what is this. i dont know` | `What is this. I do not know.` |

##  Punctuation

| Before | After |
|--------|-------|
| `hello there` | `Hello there.` |
| `what is this` | `What is this?` |
| `how are you` | `How are you?` |
| `hello.how are you` | `Hello. How are you.` |

##  Filler Word Removal (Professional Mode)

| Before | After |
|--------|-------|
| `this is very really good` | `This is good.` |
| `I basically think its great` | `I think it is great.` |
| `its literally amazing` | `It is amazing.` |
| `just simply do it` | `Do it.` |

##  Complete Examples

### Example 1: Casual Chat
**Before:**
```
hey can u help me im really confused about this
```
**After:**
```
Hey can you help me I am confused about this.
```

### Example 2: Grammar Mistakes
**Before:**
```
he dont know what to do. they is confused
```
**After:**
```
He do not know what to do. They are confused.
```

### Example 3: Spelling + Grammar
**Before:**
```
i recieved a email yesterday. it was very intresting
```
**After:**
```
I received an email yesterday. It was interesting.
```

### Example 4: Professional Email
**Before:**
```
im writing to let u know that im very interested in this position. i think im a great fit
```
**After:**
```
I am writing to let you know that I am interested in this position. I think I am a great fit.
```

### Example 5: Questions
**Before:**
```
what is the best way to do this. how can i improve
```
**After:**
```
What is the best way to do this? How can I improve?
```

##  Modes

### Professional Mode (Default)
- Expands all contractions
- Removes filler words
- Fixes all grammar
- Most formal output

### Concise Mode
- Expands contractions
- Aggressively removes filler words
- Removes "I think", "maybe", etc.
- Shortest output

### Casual Mode
- Light corrections only
- Keeps some informal language
- Less aggressive
